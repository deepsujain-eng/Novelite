# Novelite

Novelite is a Kindle-inspired online book platform built using HTML, CSS, and JavaScript.

It provides an interactive reading experience where users can:

- 📖 Browse a large collection of books
- 🎧 Listen to audiobooks with text-to-speech narration
- 🎙️ Explore book-related podcasts
- 🛒 Add books to a shopping cart
- 💳 Simulate checkout with UPI-based payment options
- 👤 Create accounts and manage profiles
- 🔍 Search books, authors, and podcasts
- 📄 Open public-domain PDF books directly
- 🌙 Customize the reading experience with themes and font controls

---

## ✨ Features

### Authentication
- User registration
- User login
- Session management using Local Storage

### Book Library
- 500+ generated books
- Genre-based filtering
- Sorting by:
  - Popularity
  - Rating
  - Price
  - Title

### Novelite Reader
- In-browser reading interface
- Adjustable font size
- Light/Dark reading modes
- Reading progress tracking
- Text-to-speech narration

### Audiobooks
- Audiobook catalog
- Browser-based narration support

### Podcasts
- Book discussion podcasts
- Author interviews
- Reading cultural content

### Shopping & Checkout
- Add books to cart
- Order management
- UPI payment simulation
- Delivery address collection

### User Profile
- Profile editing
- Favorite genre selection
- Reading statistics

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/novelite.git
```

2. Open the project folder

3. Launch `Novelite.html` in your browser

No additional installation is required.
